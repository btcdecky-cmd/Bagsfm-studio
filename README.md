# Bagsfm Studio

Bagsfm Studio serves as a command center for Solana builders, offering functionalities like tracking deployments, monitoring on-chain events via Helius, and integrating with an AI-driven assistant.

## Features
- Deployment Tracking
- Monitoring On-chain Events
- AI-driven Assistant Integration

## Getting Started
Install the dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
